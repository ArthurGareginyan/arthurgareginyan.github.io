My Custom Styles PRO

Premium WordPress Plugin

"My Custom Styles PRO" is a premium WordPress plugin that gives you the ability to easily and safely add your custom styles (CSS code) to WordPress website's <head> section, directly out of your WordPress Dashboard, without any need of an external editor. This is a "PRO" version of the plugin "My Custom Styles".

 

Itâ€™s purpose is to provide a familiar experience to WordPress users. No need anymore to editing a files of your theme and/or plugins for to add custom CSS code. Just add your CSS code in the field on the plugin page and this plugin will do the rest for you.

 

It's really useful in case of any theme/plugin update, because your custom CSS code would never be overwritten. Your custom CSS code will keep on working, no matter how many times you upgrade or switch your theme.

 

On the plugin page you find the CSS editor powered by CodeMirror. It have syntax highlighting and line numbering options. Also this editor support a tab indentation.

 

Features

 

  • Responsive & Mobile optimized settings page
  • CSS minifier
  • Automatic backup all snippets to a file
  • Trigger to temporary disable the snippet
  • Button for show/hide the field with custom code
  • Syntax highlighting (by CodeMirror)
  • Line numbering
  • Active line highlighting
  • Input areas allow for tab indentation
  • Ready for translation (.pot file included)
  • Russian translation
  • Well documented
