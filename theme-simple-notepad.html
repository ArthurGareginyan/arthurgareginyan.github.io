Simple Notepad

Premium Website Template

Inspired by the idea of anarchy, I designed this premium website template for your personal blogs and diaries. This template is perfect for writers and artists.

 

WordPress version of this template can be easily customised and already translated to different languages. It utilises latest HTML-5, CSS-3, PHP and WordPress native functions for creating the awesomeness that looks good on every browser.

 

Features

 

  • Built-in Thumbnail of post
  • Built-in Pagination
  • Built-in Breadcrumbs
  • Built-in Scroll-to-Top button
  • Block “About the Author” below posts
  • Free Google Fonts used

 

Enjoy!
