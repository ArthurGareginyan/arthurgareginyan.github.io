Info for users of the 'Anarcho Notepad' theme

The "Author's Notepad" theme differs from the "Anarcho Notepad" theme in the following details:

 

  • Responsive/Mobile-Friendly layout - Theme optimized for mobile devices. This means that visitors of your website who uses smart phones or tablets will be see an adapted version of theme, so they will be able to comfortably enjoy the content of your website. This will boost your Google rankings, the number of visitors to your website, and ultimately the frequency of return visits and social shares.

 

  • Much improved design - It has the same design that the "Anarcho Notepad" theme, but much improved. Your website visitors will definitely love this design.

 

  • Lightweight - It has a more lightweight and fast loading code and graphics than the "Anarcho Notepad" theme.

 

  • WooCommerce supported -  It compatible with the WooCommerce plugin. This makes it perfect for anyone who wants to set up an e-commerce site.

 

  • Replaceable labels - It has an options in customizer for change the text of following labels: “Search”, “Pages”, “What is this place”, “Friends & Links” and “Recent Posts”. So you can replace them with any text you want.

 

  • All 732 Google Fonts - You can customize the readability of theme by choosing from any of the 732 fonts included in Google’s font library.

 

  • Yellow Sticker can be disabled - It has an option in customizer to remove the Yellow Sticker from the sidebar area, so you can place there any other sidebar widgets.

 

  • My text of copyright can be removed - It has an option in customizer to hide my text of copyright that looks like this: 'Theme "Anarcho Notepad" designed by Arthur Gareginyan.' and located in the footer area.
